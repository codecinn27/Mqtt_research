# ESP32 MQTT Example

This project demonstrates how to use an ESP32 to send MQTT messages to a Mosquitto MQTT broker and read those messages using MQTTX.

## Requirements

- ESP32 board
- Arduino IDE
- Mosquitto MQTT Broker
- MQTTX (or any other MQTT client)
